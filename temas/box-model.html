Modelo de caja CSS

Es la forma en que CSS ve a los elementos HTML y ,¿como los ve?, como si fueran cajas con las siguientes propiedades:

  1. Content

    El contendido (texto) del elemto html, tomando en cuenta sus dimenciones (width & heigth)

  2. Border

    El borde que demilita cada uno de los elementos html

  3. Padding

    El relleno, son las distancias internas (la distancia entre el elemento html y sus elementos hermanos o padres)

  4. Margin

    son las distancias externas (la distancia entre el elemento html y sus elementos hermanos o padres )

Dentro del modelo de caja hay que considerar que una caja tiene 4 lados

  1. Arriba (top)
  2. Derecha (rigth)
  3. Abajo (buttom)
  4. Izquierda (left)

Cajas de linea vs cajas de bloque

Caja de Liena

  • Ocupan el espacio necesario para mostrar su contenido
  • No tienen dimenciones modificables (width,heigth)
  • Permiten otros elementos a su lado
  • Padding y Margin solo empujan a elementos adayacentes en horizontal, NUNCA EN VERTICAL

Caja de Bloque

  • Ocupan todo el ancho disponible
  • Tienen dimenciones modificables (width, heigth)
  • No Permiten otros elementos a su lado (aunque especifique un ancho, siguen ocupando todo el espacio disponible)
